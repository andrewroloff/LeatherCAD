LeatherCAD Documentation

Welcome to LeatherCAD

LeatherCAD is a precision 2D design tool built specifically for leatherworkers. Create clean patterns, control measurements, and refine geometry with tools designed for real-world crafting.

Whether you're laying out a wallet, belt, or custom pattern — LeatherCAD keeps your workflow fast and accurate.

Core Workflow

  • Create geometry (lines, shapes)
  • Refine using snapping and guides
  • Select and adjust nodes
  • Export or print your pattern

Mouse Controls

  • Left Click: Place points or select nodes
  • Click + Drag: Box select or create shapes
  • Double Click: Select full edge/path

Keyboard Essentials

  • Esc Cancel current action
  • Enter Finish line / shape
  • Shift Enable snapping + multi-select

Tools

  • L → Line tool (pattern edges)
  • R → Rectangle (panels, pockets)
  • C → Circle (holes, rounded cuts)
  • N → Node tool (fillet / edge smoothing)
  • Space → Selection tool

Each tool is designed to match common leather pattern tasks.

Selection & Editing

  • Click a node to select it
  • Shift + Click for multi-select
  • Drag to box select multiple nodes
  • A Select all

Move (Transform)

  • Select nodes
  • Press G to move
  • Move mouse to adjust
  • Click to confirm placement
  • Esc cancels movement

Axis Lock:

  • X Lock horizontal movement
  • Y Lock vertical movement

Snapping

  • Hold Shift to enable snapping
  • Snap to:
    • Grid (consistent spacing)
    • Nodes (precise joins)
    • Guides (layout alignment)

Snapping helps ensure clean joins and accurate measurements — critical for pattern cutting.

Guides

  • H Horizontal guide
  • V Vertical guide
  • Drag guides to reposition

Use guides to plan stitch lines, spacing, and alignment before committing geometry.

Node Tool (Edge Rounding / Fillet)

  • Select Node tool (N)
  • Click a corner node
  • Drag to create a rounded edge
  • Release to apply

Perfect for smoothing corners on wallets, straps, and panels.

File & Output

  • New: Start fresh pattern
  • Open: Load saved design
  • Save As: Export pattern file
  • Print: Print full-scale pattern

Patterns are designed to print at accurate real-world scale.

Undo / Redo

  • Ctrl + Z Undo
  • Ctrl + Shift + Z Redo

Workflow Tips

  • Use guides before drawing complex layouts
  • Snap to nodes to avoid tiny gaps in patterns
  • Round corners early for cleaner designs
  • Use move tool for precise adjustments instead of redrawing
  • Press Esc multiple times to fully exit actions

About LeatherCAD

LeatherCAD is built for makers who want precision without complexity. Focused on pattern design, it removes unnecessary CAD overhead and keeps you in control of your craft.
