# LeatherCAD

An open-source 2D CAD tool built with Electron for Windows, macOS, and Linux.  
Supports drawing polylines, rectangles, circles, guides, and customizable grids.

## Features

- Real-time grid snapping (`in`, `mm`, `cm`)
- Polyline, rectangle, circle tools, fillets/bevels
- Undo/redo history
- Guides and guide snapping
- Dark mode support
- Cross-platform

## Installation

### Run from source
```bash
git clone https://github.com/andrewroloff/leatherCAD.git
cd leatherCAD
npm install
npm start
